# Database (scripts SQL)

Ce dossier sert a partager les scripts SQL du projet (schema, donnees de test, migrations manuelles).

Contenu propose:
- `schema.sql` : creation des tables/contraintes
- `seed.sql` : donnees de test (optionnel)

Convention:
- Garder les scripts idempotents quand c'est possible (ou documenter les pre-requis).
- Mettre une courte description en tete de fichier.
